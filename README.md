# Restaurant System 🍽

A simple restaurant management system built using HTML, CSS, and JavaScript.

## Features
- Food ordering system
- Table reservation system
- Admin dashboard
- Inventory management
- Update food prices
- Add and remove admins
- Local storage support

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download the project files
2. Open `restaurant.html` in your browser
